# 生态项目协作系统

一个完整的企业生态协作管理平台，用于管理从需求捕获到伙伴匹配的全流程合作业务。

## 项目简介

本系统旨在帮助企业管理其生态合作伙伴关系，通过智能匹配算法和工作流可视化，提升项目协作效率。系统支持需求管理、产品分析、伙伴管理、伙伴匹配和流程跟踪等核心功能。

## 功能特性

### 核心功能

- **需求管理**：捕获和管理客户项目需求，跟踪需求状态
- **产品分析**：对需求进行可行性、复杂度和优先级分析
- **伙伴管理**：维护合作伙伴档案，管理其能力和资质信息
- **智能匹配**：基于能力评分算法，为项目推荐最合适的合作伙伴
- **流程跟踪**：可视化展示从需求到落地的完整协作流程

### 系统亮点

- 📊 **实时仪表板**：数据驱动的决策支持
- 🎯 **智能推荐**：多维度伙伴匹配算法
- 📈 **可视化图表**：Canvas 绘制的动态数据展示
- 🎨 **现代界面**：暗色主题，响应式设计
- 📝 **活动审计**：完整的操作日志追踪

## 快速开始

### 安装

本项目为纯前端应用，无需构建步骤，直接打开即可使用：

```bash
# 克隆项目
git clone <repository-url>

# 进入项目目录
cd ecosystem-collaboration

# 在浏览器中打开 index.html
# 或使用本地服务器
python -m http.server 8000
# 然后访问 http://localhost:8000
```

### 使用说明

1. **需求录入**：在"需求管理"页面创建新的项目需求
2. **产品分析**：对需求进行分析评估
3. **伙伴管理**：添加和管理合作伙伴信息
4. **伙伴匹配**：系统自动为项目推荐合适的伙伴
5. **流程监控**：在仪表板查看整体协作进度

## 数据模型

系统管理以下核心实体：

| 实体 | 说明 | 字段示例 |
|------|------|----------|
| **需求** | 客户项目需求 | 标题、描述、预算、截止日期 |
| **分析** | 产品分析结果 | 可行性、复杂度、优先级评分 |
| **伙伴** | 合作伙伴档案 | 名称、行业、能力标签、评分 |
| **匹配** | 伙伴-项目匹配 | 匹配度分数、推荐理由 |
| **活动** | 系统操作日志 | 操作类型、时间、操作人 |

## 界面预览

系统采用暗色主题设计，主要界面包括：

- **侧边栏导航**：快速切换不同功能模块
- **仪表板**：关键指标和趋势图表
- **需求列表**：卡片式展示所有需求
- **伙伴库**：合作伙伴档案管理
- **匹配中心**：智能匹配推荐结果

## 开发说明

### 代码结构

- **app.js** 分为以下几个模块：
  - Data Store：数据存储和管理
  - Utilities：通用工具函数
  - Navigation：页面导航逻辑
  - Modals & Drawers：弹窗和抽屉组件
  - Entity Renderers：各实体渲染函数
  - Chart Drawing：图表绘制函数
  - Form Handlers：表单处理逻辑

### 扩展开发

如需添加新功能，参考以下步骤：

1. 在 `app.js` 中定义数据模型
2. 创建渲染函数（render*）
3. 添加导航处理逻辑
4. 在 `index.html` 中添加 UI 结构
5. 在 `style.css` 中添加样式
